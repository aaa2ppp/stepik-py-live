Эта форма создает новую случайную жизнь на поле заданного размера. Допустимые значения размеров от 1 до 100.

Остальные поля устанавливают начальные параметры запроса /live?<query>:

serial=<int>
номер поколения начиная с 0 (если выбрать большое значение совместно с большим размером поля игры, расчет истории поколений может занять длительное время);
autoupdate={on|off}
включить авто обновление, по умолчанию off;
update_period=<int>
период обновления в ms, минимальное допустимое значение 100 (при малых значениях, слабом процессоре, будут просадки);
js={on|off}
включает/отключает JavaScript на стороне клиента, по умолчанию on (если выключить, то авто обновление работать не будет).

Вы можете изменять их в адресной строке в любое время, после создания жизни. Не забывайте нажать Enter после изменения. Кнопка Остановить/Продолжить управляет autoupdate. Для остальных параметров гуя нет.
